这是关于场景任务脚本的文档。从目前起，我们将使本文档保持最新状态。对于具有所需和验证布局的 quest 文件，我们将提供转换工具来更新此功能。

Saga.AddStep


Saga.AddStep（cid， 任务ID）;


言论：


向任务添加步骤。这只能在初始化期间完成。


例：


下面的代码示例说明了 Saga.AddStep 的用法：

function QUEST_START(cid)
-- Start Quest
Saga.AddStep(cid, QuestID, 101);
Saga.QuestStart(cid, QuestID);
return 0;
end

Saga.GetStepIndex

Saga.GetStepIndex(cid, QuestId);


言论：


返回任务的当前步骤。


例：


下面的代码示例说明了 Saga.GetStepIndex 的用法：

function QUEST_CHECK(cid)
-- Check all steps for progress
local CurStepID = Saga.GetStepIndex(cid, QuestID );
local ret = -1;

if CurStepID == 101 then
  ret = QUEST_STEP_1(cid);
end

if ret == 0 then
  QUEST_CHECK(cid)
end

return -1;
end

Saga.QuestComplete

Saga.QuestComplete(cid, QuestID);


言论：


完成场景任务并删除与任务相关的任何相关信息。


例：


以下代码示例说明了 Saga.QuestComplete 的用法：

function QUEST_STEP_1(cid)
   Saga.StartEvent(cid, 10102);	
   Saga.StepComplete(cid, QuestID, 101);
   Saga.QuestComplete(cid, QuestID);
   return -1; 
end

Saga.QuestStart

Saga.QuestStart(cid, QuestId);


言论：


使用指定的任务 ID 启动并初始化场景任务。


例：


以下代码示例说明了 Saga.QuestStart 的用法：

function QUEST_FINISH(cid)
-- Start Next Quest
Saga.QuestStart(cid, 2);
return 0;
end

Saga.StartEvent

Saga.StartEvent(cid, EventId);


言论：


开始显示事件对话框。它将暂停函数，直到事件结束。


例：


下面的代码示例说明了 Saga.StartEvent 的用法：

function QUEST_STEP_1(cid)
   Saga.StartEvent(cid, 10102);	
   Saga.StepComplete(cid, QuestID, 101);
   Saga.QuestComplete(cid, QuestID);
   return -1; 
end

Saga.StepComplete

Saga.StepComplete(cid, QuestID, StepID);


言论：


完成指定步骤并前进到下一步。如果未指定任何步骤，则下一步被视为 0。


例：


下面的代码示例说明了 Saga.StepComplete 的用法：

function QUEST_STEP_1(cid)
   Saga.StartEvent(cid, 10102);	
   Saga.StepComplete(cid, QuestID, 101);
   Saga.QuestComplete(cid, QuestID);
   return -1; 
end
