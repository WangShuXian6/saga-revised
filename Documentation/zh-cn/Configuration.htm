类型扩展：

默认情况下，Saga使用默认的文件格式xml或csv。但有时你可能有自己的文件格式，或者想添加加密功能。不要担心，Saga支持通过脚本扩展类型，你可以编写自定义覆盖类型并将其用作服务器的基础。

你可以在配置文件中使用"type"参数来指定类型重载。

示例： 
<Saga.Factory.Addition type="Plugins.ExtendedAddition">
	<Files>
		<add path="../Data/addition.csv" />
	</Files>
</Saga.Factory.Addition>

Saga.Factory.Addition。

概述

Addition工厂加载大量数据，定义某些物品和技能对玩家状态的副作用。大多数增益/减益效果和物品都以此为基础。Addition工厂默认情况下没有其他选项，除了分片文件加载。

配置： 

<configuration>
	<configSections>
		<section name="Saga.Factory.Addition" type="Configuration.Factory_Addition, Saga.Map"/>		
	</configSections>
	
	<!-- 这里是其他配置选项 -->
	
	<Saga.Factory.Addition>
		<Files>
			<add path="../Data/addition.csv" />
		</Files>
	</Saga.Factory.Addition>
</configuration>

Saga.Factory.Exptable。

概述

经验表加载数据，定义每个等级所需的经验值。请注意，经验表与客户端一起工作，如果经验表有误，可能会导致游戏体验不佳。

附加参数

cexp 你可以使用cexp来设置速率修正符。这相当于exp = exp_reward * rate;
jexp 你可以使用jexp来设置速率修正符。这相当于exp = exp_reward * rate;
wexp 你可以使用wexp来设置速率修正符。这相当于exp = exp_reward * rate;

配置： 

<configuration>
	<configSections>
		<section name="Saga.Factory.Exptable" type="Configuration.Factory_Exptable, Saga.Map"/>		
	</configSections>
	
	<!-- 这里是其他配置选项 -->
	
	<Saga.Factory.Exptable [cexp="整数"] [jexp="整数"] [wexp="整数"]>
		<Files>
			<add path="../Data/expierence.csv" />
		</Files>
	</Saga.Factory.Exptable>
</configuration>

Saga.Factory.Itemdrops。

概述

物品掉落管理器不是一个关键组件。它被添加的目的是让测试服务器能够快速修改物品掉落，并按需重新加载。也可以作为通用数据库提取的一部分，使我们保持数据的最新。

配置： 

<configuration>
	<configSections>
		<section name="Saga.Factory.Itemdrops" type="Configuration.Factory_Itemdrops, Saga.Map"/>		
	</configSections>
	
	<!-- 这里是其他配置选项 -->
	
	<Saga.Factory.Itemdrops>
		<Files>
			<add path="../Data/item_drops.csv" />
		</Files>
	</Saga.Factory.Itemdrops>
</configuration>

Saga.Factory.Items

概述

Item工厂是一个非常重要的类。它包含所有物品的信息：拍卖的类别、售价、物品是否可交易，及其适用于哪些职业。

配置： 

<configuration>
	<configSections>
		<section name="Saga.Factory.Items" type="Configuration.Factory_Items, Saga.Map"/>		
	</configSections>
	
	<!-- 这里是其他配置选项 -->
	
	<Saga.Factory.Items>
		<Files>
			<add path="../Data/item_data.xml" reader="xml"  />
		</Files>
	</Saga.Factory.Items>
</configuration>

Saga.Factory.Portals

概述

门户工厂包含地图上的所有传送门。传送门定义在地图上，作用是让玩家跳跃到其他区域或特殊区域。

配置： 

<configuration>
	<configSections>
		<section name="Saga.Factory.Portals" type="Configuration.Factory_Portals, Saga.Map"/>		
	</configSections>
	
	<!-- 这里是其他配置选项 -->
	
	<Saga.Factory.Portals>
		<Files>
			<add path="../Data/portals.csv" />
		</Files>
	</Saga.Factory.Portals>
</configuration>
