这是Saga身份验证服务器的命令行界面文档参考。

按字母排序的功能

  1. account -create
  2. account -ip
  3. account -isonline
  4. account -kick
  5. account -time
  6. acl -add
  7. acl -dump
  8. acl -remove

account -create

备注：

此命令将创建一个常规账户，且无GM等级。

示例：

以下代码示例演示了如何使用account -create：

Server instance
account -create phr34k918 ha2jumi female
account created

account -ip

备注：

显示玩家最后使用的IP地址。

示例：

以下代码示例演示了如何使用account -ip：

Server instance
account -ip phr34k
User last ip address: 127.0.0.1
account -ip phr34k918
User last ip address: 0.0.0.0

account -isonline

备注：

检查指定账户是否当前在线。

示例：

以下代码示例演示了如何使用account -isonline：

Server instance
account -isonline phr34k918
User is: offline

account -kick

备注：

此命令将查找您的活动会话ID。如果该字段非零，它将联系最后一次玩的世界服务器来踢出您的会话。

示例：

以下代码示例演示了如何使用account -kick：

Server instance
account -kick phr34k

account -time

备注：

计算用户实际游戏时间。如果用户当前在线，添加的游戏时间将不会更新。

示例：

以下代码示例演示了如何使用account -time：

Server instance
account -time phr34k
User has played: 0 days - 0 hours - 6 minutes
account -time phr34k918
User has never played

acl -add

备注：

此命令将向访问控制列表添加规则策略。

示例：

以下代码示例演示了如何使用acl -add：

Server instance
acl -add allow 0.0.0.0/0
Entry added

acl -dump

备注：

此命令将导出被禁止的IP和过滤掩码列表。您还可以选择使用IP地址作为第三个参数来检查哪些掩码影响指定的IP地址。它会在与二进制文件相同的目录中创建一个acl.txt文件。

示例：

以下代码示例演示了如何使用acl -dump：

Server instance
acl -dump 127.0.0.1
Output generated in: acl.txt
Access Control Lists
===================================================================================
RULE#   ALLOW/DENY           IP              MASK           
1   DENY                 0.0.0.0         0.0.0.0        
2   ALLOW                127.0.0.1       255.255.255.255

acl -remove

备注：

此命令将删除先前添加到访问控制列表中的规则策略。

示例：

以下代码示例演示了如何使用acl -remove：

Server instance
acl -remove allow 0.0.0.0/0
Entry removed
