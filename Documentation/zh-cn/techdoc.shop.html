文件格式： Shoplist


warper 文件格式用于配置 Npcfunction 'ShopConversation'。供有商店的 NPC 使用。

<?xml version="1.0" encoding="utf-8" ?>
<!DOCTYPE shop [
<!ELEMENT shop (item*)>
<!ELEMENT item CDATA #REQUIRED>
<!ATTLIST item count CDATA #IMPLIED>
]>
<shop>
  <item>3012068</item>
  <item>3012067</item>
  <item>3012066</item>
  <item>52</item>
  <item>3000001</item>
</shop>


在上面的示例中，您将看到用于配置 ShopConversation 元素的内部文件格式。这将显示一个普通商店，您可以在其中购买、出售或重新购买您的物品。商店不仅支持无限数量的供应，还可以设置为库存。


一旦你使用 buy a stocked item，NPC 的库存数量就会减少。一旦它达到零，该物品就会从列表中删除，直到 npc 被重新加载。默认情况下，服务器每小时都会重新配置其 npcs silent。


为了更改商店出售该商品的商品的价格，您需要修改客户端文件中的items.dat。我们建议不要更改客户端，因为这会使您的服务器更新变得非常复杂。要使用库存项目，请将 count 属性配置为大于 0，这将导致系统不使用无限库存。
